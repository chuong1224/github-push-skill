## English

### Overview

This skill helps you quickly create a new GitHub repository, push files or code, and set up a professional **bilingual (English + Vietnamese)** README with badges and Version History.

**Trigger phrases:**
- "push to github"
- "tạo repo và push"
- "github push"
- "setup github repo"
- "create repo"

### Features

- Automatic GitHub repo creation
- Bilingual README (English on top, Vietnamese below)
- Professional header badges
- Standardized README structure
- Version History table
- Smart handling when repo already exists
- Uses Conventional Commits

### Workflow

1. Define requirements (name, description, project type, visibility)
2. Create repository
3. Prepare files (README.md, LICENSE, .gitignore, SKILL.md if needed)
4. Write bilingual README following the required structure
5. Push files using `github___push_files`
6. Return repo link and ask for next steps

## Tiếng Việt

### Tổng quan

Skill này giúp bạn nhanh chóng tạo repository GitHub mới, push code/files và setup README song ngữ Anh-Việt chuyên nghiệp, có badge và Version History.

**Cụm từ kích hoạt:**
- "push to github"
- "tạo repo và push"
- "github push"
- "setup github repo"
- "create repo"

### Tính năng

- Tự động tạo repo GitHub
- README song ngữ (Tiếng Anh ở trên, Tiếng Việt ở dưới)
- Header badge chuyên nghiệp
- Cấu trúc README chuẩn hóa
- Bảng Version History
- Xử lý thông minh khi repo đã tồn tại
- Sử dụng Conventional Commits

### Quy trình

1. Xác định yêu cầu (tên repo, mô tả, loại project, public/private)
2. Tạo repository
3. Chuẩn bị file (README.md, LICENSE, .gitignore, SKILL.md nếu cần)
4. Viết README song ngữ theo cấu trúc bắt buộc
5. Push file bằng `github___push_files`
6. Trả link repo và hỏi bước tiếp theo
